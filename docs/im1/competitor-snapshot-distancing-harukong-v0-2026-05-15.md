## 0. 한 줄 결론

> **디스턴싱(CBT 코치 동반·고가·치료 포지셔닝)과 하루콩(이모티콘 기분 일기·시각화·대중 무료)은 BlueBird 6축에서 모두 다른 위치를 차지한다. BlueBird의 비어 있는 칸 = "분석 도구·자율성·중간 가격·운영자 자기상".**

## 1. 6축 비교 매트릭스

| 축 | **BlueBird** (현재) | **디스턴싱** | **하루콩** |
|---|---|---|---|
| **톤** | 분석가·관찰자 (위로·응원 차단) | 치료·전문가 (CBT 코치) | 친근·캐릭터 (콩 이모티콘) |

## 2. 디스턴싱 — 상세

### 2.1 포지셔닝 (공식 사이트·App Store 기준)

- 한 줄: "**인지행동치료 기반 정신건강 프로그램**" / "**나를 바꾸는 마음의 기술**"
- 카피 어휘: "마음 문제 해결", "멘탈케어", "마음건강", "심리상담", "우울증 앱"
- 이론적 기반: CBT + ACT + MBCT 명시
- 효과 광고: "우울감 34% ↓ · 불안감 40% ↓"

### 2.2 차별화 vs BlueBird

| 항목 | 디스턴싱 | BlueBird |
|---|---|---|
| 치료/도구 구분 | **치료 (의료 어휘)** | **분석 도구 (도구 어휘)** — 카테고리 가드 (`scripts/lint-copy.ts` MEDICAL_PATTERNS) |
| 가격대 | 주 ~30k, 월 80k~295k | 미정. 인터뷰 후 결정 |
| 운영 형태 | 코치 동반 (인력 비용) | AI 분석 (Gemini API, 비대면) |
| 효과 약속 | "34% ↓ / 40% ↓" 정량 약속 | 효과 보장 안 함 (학술 인용은 배경 정보만 — `app/disclaimer`) |
| 익명성 | 100% 익명 광고 | 가입 필요 (Supabase Auth) |

### 2.3 본질 위협 측면

**위협 #1 (사용자가 "치료받고 싶다"고 surface):** 디스턴싱이 치료 포지셔닝을 점유하고 있으므로, BlueBird 사용자가 "디스턴싱이 더 나아 보인다"고 응답 시 분류 케이스:
- IM.1 코딩 E1 정합 + 비교 차이 항목 기록
- 누적 ≥10% → CSO escalation (도구 vs 치료 포지셔닝 재검토)

**위협 시그널 가설 (사용자 발화):**
- "치료가 필요한 거면 디스턴싱이 낫지 않나?" — 본질 위협 #1
- "코치가 매일 봐주는 게 좋아요" — 본질 위협 #2 (인적 동반 회귀)

### 2.4 가격 관점

디스턴싱은 ~월 80k+ 고가 모델로 운영. 이는 1인 운영 + AI 자동화 BlueBird가 직접 경쟁할 segment가 아님. **BlueBird 잠재 가격대 추정: 월 5~15k (PMF plan §0 결제 의향 ≥20% 게이트 인터뷰 후 정밀화).**

## 3. 하루콩 — 상세

### 3.1 포지셔닝 (Google Play·App Store)

- 한 줄: "**하루를 기록하는 가장 간단한 방법**"
- 카피 어휘: "콩 모양 이모티콘", "기분 기록", "월별 캘린더", "감정의 흐름 시각화"
- 글로벌 성과: 전세계 누적 1천만 DL, 해외 사용자 80%+ (2025 유니콘팩토리)

### 3.2 차별화 vs BlueBird

| 항목 | 하루콩 | BlueBird |
|---|---|---|
| 입력 형태 | 이모티콘 선택 (5종 색·표정) | 자동 사고 텍스트 입력 |
| 출력 형태 | 시각화 (월/주 캘린더·통계) | 왜곡 분류 + Δpain·autonomy 시계열 |
| 사고 처리 | 없음 (감정 라벨링만) | 분석 (Gemini multi-distortion + 소크라테스 + 행동) |
| 시간 모델 | 매일 1회 (한 줄·사진 추가 가능) | 짧은 체크인 + 분석·소크라테스·행동 4-단계 funnel |
| 가격 | 무료 + 일부 유료 | 미정 |
| 자기상 | 기록자 ("나의 감정 흐름") | 운영자 ("나의 사고 운영체계") |

### 3.3 본질 위협 측면

**위협 #4 (사용자가 "캐릭터 친근체" 요구):** 하루콩은 콩 이모티콘·친근 톤으로 대중에게 점유됨. BlueBird 사용자가 "더 귀여웠으면" 발화 시:
- IM.1 코딩 D4 정합
- 누적 ≥10% → CSO escalation (자기상 운영자 vs 기록자 재검토)

**위협 시그널 가설:**
- "콩 같은 캐릭터가 있으면 좋겠어요" — 본질 위협 #4
- "감정만 기록하는 게 편해요. 분석은 부담돼요" — 1차 타겟 3축 적중률 하락 시그널

### 3.4 가격·규모 관점

하루콩이 보여주는 것: **무료 + 시각화** 조합은 대중 도달 1천만 DL 가능. 그러나 **결제 의향 측정 어려움** (수익 모델 광고·일부 유료 추정). BlueBird는 의도적으로 **유료 인터뷰부터 검증**하므로 하루콩과 경쟁 segment 다름.

## 4. 비어 있는 칸 — BlueBird의 기회

6축 매트릭스에서 두 경쟁사가 점유하지 않은 위치:

| 위치 | 의미 | BlueBird 정합 |
|---|---|---|
| **분석 도구** (치료 X) | 카테고리·법적·운영 비용 모두 낮음 | ✓ 정합 |
| **운영자 자기상** | "내가 내 사고를 운영" — 디스턴싱(환자)·하루콩(기록자) 모두 점유 안 함 | ✓ 정합 |
| **AI 자동 분석** | 인력 비용 0, 24/7 가능 | ✓ 정합 |
| **중간 가격** (월 5~15k 추정) | 디스턴싱 고가·하루콩 무료 사이의 공백 | ⚠️ PMF 인터뷰 후 결정 |
| **디버깅 메타포** | 의료(디스턴싱)·일상(하루콩) 어휘와 다름. 1차 타겟 3축 (인지 스타일·분석가형)에 강한 정렬 | ✓ 정합. 글로벌 챗봇 한국어 진입에도 차별화 자산 |

CSO 차별화 3축 정합성 점검에 다음 신호 추가:

| 신호 | 출처 | 임계 |
|---|---|---|
| 디스턴싱 가격대 변동 | App Store 가격 페이지 | 월 30k 미만 진입 시 → 가격 경쟁 시그널 |
| 디스턴싱 새 카테고리 출시 (AI 분석·운영자 자기상) | 공식 사이트·언론 | 1건+ 시 즉시 CSO escalation |
| 하루콩 분석 기능 추가 | 공식 사이트·App Store changelog | "분석", "AI", "왜곡" 어휘 추가 시 escalation |
| 글로벌 챗봇 한국어 진입 (ChatGPT·Claude·Gemini "감정 코치" 모드) | 자체 모니터링 | 1건+ 시 즉시 escalation |
| IM.1 인터뷰 E1 정합률 (디스턴싱 비교 언급) | IM.1 D14 코딩 | ≥10% 시 카피·자기상 재검토 |
| IM.1 인터뷰 D1·D4 정합률 (위로·캐릭터 회귀) | 동일 | ≥10% 시 본질 위협 hotfix |

각 신호는 다음 weekly all-hands(2026-W21+)에 carry-over.

## 6. v0 → v1 격상 조건

본 v0는 **공개 자료(공식 사이트·App Store·언론)만으로 작성**된 1차 캡처. 다음 시점에 v1으로 격상:

- IM.1 D14 인터뷰 코딩 결과 (E1·D1·D4) 반영
