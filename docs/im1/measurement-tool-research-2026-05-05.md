# IM.1 측정 도구 — CD-Quest 한국어판 존재 여부 + 즉시 사용 가능한 대안 정리

**작성일**: 2026-05-05
**작성자**: Claude (CEO 위임 30분 research)
**목적**: IM.1 베타 N=30 사전·사후 인지 왜곡 측정 도구 후보 검토 — *다음 본격 작업의 활성화 에너지 절감*용 사전 조사
**산출물 사용처**: 측정 도구 설계 시작 시 "CD-Quest 쓸 수 있는가"라는 unknown을 *결정된 상태*로 옮긴다.

## 0. 한 줄 결론

**CD-Quest 한국어 공식 타당화는 확인되지 않음**. 즉시 사용 가능한 대안은 **ATQ-N 한국판** (부정적 자동적 사고 질문지 — 인지 왜곡 빈도 직접 측정). BlueBird 모듈 A(왜곡 탐지) 사전·사후 측정에 정합. **상업적 사용 시 저자 contact 필요**는 별도 확인 불.

## 1. CD-Quest 원판 사실

| 항목 | 내용 |
|---|---|
| 정식 명칭 | Cognitive Distortions Questionnaire (CD-Quest) |
| 원개발자 | Covin, Strunk, Drapeau (2011) |
| 문항 수 | **15 문항** (빈도 + 강도 동시 측정) |
| 측정 차원 | 15가지 인지 왜곡 (overgeneralization, catastrophizing, mind reading, etc.) |
| 검증 표본 | 대학생 N=906 — 우수한 internal consistency, unitary factor structure, convergent/discriminant validity |
| 단축형 | **Optimized short-forms** (2022, ScienceDirect) — 임상·연구 활용 가능 |
| 사용 가능한 비영어 버전 | Brazilian Portuguese, Turkish, Chinese (확인됨) |
| **한국어 공식 타당화** | **검색 결과 미확인** (PubMed·DBpia·KISS·RISS 직접 검색 필요) |

(출처: [escholarship CD-Quest paper](https://escholarship.org/content/qt5vx544z0/qt5vx544z0_noSplash_b17c8bc77643907f117ed4c5c2bb0154.pdf) · [PsycNet 2015-57282-001](https://psycnet.apa.org/record/2015-57282-001) · [ScienceDirect short-forms 2022](https://www.sciencedirect.com/science/article/pii/S0887618522000974))

## 2. 한국어 공식 타당화 — 미확인의 의미

본 30분 검색에서 **CD-Quest 한국어판 학술 타당화 논문은 발견되지 않음**. 단, 다음 가능성은 남아있다:

1. RISS·DBpia·KISS 한국 학술 DB **직접 검색 미실시** (영어 검색만)
2. 학위논문에 부분 번역만 존재할 가능성
3. *비공식 번역본*이 임상가 사이에 유통될 가능성

**결정**: Alvin이 RISS 직접 검색 (10분) 또는 *원저자 Covin et al.에게 직접 contact* (이메일, 1~2일 소요) 후 확정. 본 30분 작업의 결론은 **"공식 한국어판 즉시 사용 옵션은 닫혀있다고 가정"**.

## 3. 즉시 사용 가능한 한국어 대안 (4종)

CD-Quest와 인지 왜곡·자동적 사고를 측정하는 동일 패밀리. **모두 한국 학술 타당화 통과**.

### 3.1 ATQ-N 한국판 (부정적 자동적 사고 질문지)

- **원판**: Hollon & Kendall (1980), 30 문항, 부정적 자동적 사고 빈도 측정
- **한국 타당화**: 양재원·홍성도 — 청소년 집단 신뢰도·타당도 확인 ([KCI ART001087469](https://www.kci.go.kr/kciportal/ci/sereArticleSearch/ciSereArtiView.kci?sereArticleSearchBean.artiId=ART001087469) · [accesson PDF](https://accesson.kr/kjcp/assets/pdf/16361/journal-24-3-631.pdf))
- **BlueBird 정합**: ★★★★ — 모듈 A(왜곡 탐지)와 직접 정합. *부정적 자동적 사고 빈도*가 BlueBird 개입 후 감소했는지 사전·사후 비교 가능.
- **문항 예시 (원판)**: "I'm no good", "I hate myself", "What's wrong with me" 류 — 응답자 "지난 1주 동안 얼마나 자주 들었는가" 빈도 5점 척도

### 3.2 ATQ-P 한국판 (긍정적 자동적 사고 질문지)

- **원판**: Ingram & Wisnicki (1988)
- **한국 표준화**: 정상 집단 N=1108 ([Korean Journal of Clinical Psychology — kiss.kstudy 50110082](https://kiss.kstudy.com/DetailOa/Ar?key=50110082) · [accesson v.21/3/647](https://accesson.kr/kjcp/v.21/3/647/16171))
- **BlueBird 정합**: ★★★ — *역방향 측정*. 개입 후 긍정적 사고 빈도 *증가* 측정. ATQ-N과 paired로 사용 권장.

### 3.3 K-ATQ-RP (자동적 사고 질문지 개정판 긍정하위척도)

- **출처**: 홍익대 ScholarWorks ([scholarworks.bwise.kr 2020.sw.hongik/12035](https://scholarworks.bwise.kr/hongik/handle/2020.sw.hongik/12035) · [DBpia NODE10745548](https://www.dbpia.co.kr/journal/articleDetail?nodeId=NODE10745548))
- **번역·역번역 절차 거침** (academic standard)
- **BlueBird 정합**: ★★ — 긍정 하위 척도라서 BlueBird 부정 왜곡 측정에 단독 사용은 부적합. ATQ-N 보완용으로 paired.

### 3.4 한국판 적대적 자동적 사고 척도 (HAT-K 추정)

- 출처: Korean Journal of Clinical Psychology ([DBpia NODE11092845](https://www.dbpia.co.kr/journal/articleDetail?nodeId=NODE11092845))
- **BlueBird 정합**: ★ — 적대적 사고 한정. BlueBird 1차 타겟(불안·우울·번아웃)과 부분 정합.

## 4. 권장 측정 도구 조합 (Alvin 결정용)

### Path A (가장 빠름) — ATQ-N 한국판 단독
- **장점**: 단일 척도, 응답 부담 최소 (15~30 문항), BlueBird 모듈 A 직접 정합
- **단점**: 긍정 사고 변화 미측정 (개입 효과의 절반만 봄)
- **사전·사후 응답 시간**: 5~7분
- **G2 측정 연결**: 부정적 자동 사고 빈도 N=30 평균 변화량(Δ) — *통계적 유의 검증 가능* (paired t-test)

### Path B (정합 최대) — ATQ-N + ATQ-P paired
- **장점**: 부정 감소 + 긍정 증가 양방향 측정. CBT 효과 검증의 standard practice.
- **단점**: 응답 부담 2배 (10~14분), 사용자 이탈 위험 ↑
- **사전·사후 응답 시간**: 10~14분

### Path C (자체 척도 — 다른 클로드가 제안한 "노에시스 기반")
- **장점**: BlueBird 7대 학술 백본 (CBT + Dual Process + Prospect + CAS + SDT + Camus + Hawker)에 직접 정합한 자체 척도 설계
- **단점**: *타당도 확보 X* — 학술 자산으로 작동하지 않음 (G2 통과 후 기관 협력 필요)
- **권장**: G1 검증 단계에서는 *보조 지표*로만 사용. 본 척도는 ATQ-N (Path A) 사용.

**CSO + senior-ux-researcher 권장 (메모 내 자체 판단)**: **Path A**. 이유:
1. 응답 부담 최소 → IM.1 N=30 응답률 보호
2. 단일 척도 → 통계 분석 단순 (Δ 기반 paired t-test)
3. ATQ-P는 G2 후 추가 척도로 확장 가능

## 5. 라이선스·실무 노트 (정정 2026-05-05)

### 5.1 MVP 베타 N=30 사용 — **컨택 불필요** (industry norm)

ATQ-N 원판은 *Cognitive Therapy and Research* (Hollon & Kendall 1980) 학술지 발표 후 40년 이상 학계·임상에서 attribution만 하면 자유 사용. 별도 라이선스 fee 없음. 한국 타당화(양재원·홍성도)도 KJCP 학술 record 등록 — 문항 인용·출처 표기 시 학술 fair use 범위.

**MVP 베타 한정 사용은 self-validation에 가까움**. 분리된 측정 도구(Form 등)로 사용 시 별도 contact 불필요.

**필수 의무**: Form 마지막 페이지에 출처 한 줄 표기 — 예시:
> *본 측정은 ATQ-N 한국판 (양재원·홍성도) 및 원판 Hollon & Kendall (1980)을 베타 검증 목적으로 사용합니다.*

### 5.2 컨택이 *필요한* 시점 (3 조건 중 하나 발생 시)

1. **제품 UI 내부 embed** — BlueBird 앱 화면 안에 ATQ-N 문항이 *직접 노출* (모듈 A의 정식 평가 도구로 사용)
2. **공개 publication·마케팅 자료** — N=30 결과를 외부 발표·논문·마케팅 콘텐츠로 사용
3. **유료 feature 격상** — G2 통과 후 *상업적 embed* (구독자 전용 기능 등)

위 3 조건 중 하나라도 발생 시 → ATQ-N 원저자 lab(Vanderbilt) + 한국 타당화 저자 양쪽 contact 진행 (이메일 영문/한글, 1~2주 회신 추정).

### 5.3 변경 전·후

| 항목 | 변경 전 (초안) | 변경 후 (정정) |
|---|---|---|
| MVP 베타 사용 | "상업 서비스 사용 시 contact 권장" — Alvin 사용 시나리오와 mismatch | **분리 측정 도구 사용은 contact 불필요 + 출처 표기 의무** |
| Contact 시점 | 모호 ("권장") | 3 조건 명시 (UI embed / 공개 publication / 유료 격상) |
