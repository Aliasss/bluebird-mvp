### 2.5 data-analyst

**Migration 15 이벤트 스키마 검토**:

14개 event_type 분류:
```
p2_checkin_prompt_shown    p2_checkin_prompt_clicked  p2_checkin_prompt_dismissed
p3_reminder_sent           p3_reminder_clicked        p3_reminder_dismissed
permission_requested       permission_granted         permission_denied
subscribed                 unsubscribed
push_sent                  push_clicked               push_dismissed
```

**측정 가능한 퍼널 (permission → engagement)**:
```
permission_requested → permission_granted (동의율)
             ↓
         subscribed (실구독 전환율)
             ↓
       push_sent → push_clicked (CTR)
```

**IM.1 측정 지원**: 권한 동의율·구독 전환율·CTR 측정 가능. PMF 퍼널 중 "알림 참여 → 체크인 재방문 → 30일 잔존" 연결 고리 데이터화 가능.

**Falsifiability 가드 — 현재 제약**:
- N=0: 베타 미시작 → 모든 메트릭 점추정 불가. 신뢰구간 불계산 구간.
- 소표본 가드 활성화: N<50 도달 전까지 push CTR 등 메트릭은 방향성 참고용으로만 사용. 통계적 결론 보류.
- 코호트 정의: 1차 타겟 3축 충족자 vs 비충족자 분리 추적 인프라 필요 (현재 미분리 — 이벤트에 cohort 필드 없음). 향후 스키마 확장 검토 권고.
