# 2026-05-19 비즈니스 모델 deep dive — CPO/CSO/CMO

**참여:** CPO · CSO · CMO + CEO
**소집 배경:** 사용자(CEO)가 외부 Claude 대화에서 받은 비평을 공유. "월 구독 × 개인 × 인지 작업" 조합의 retention 구조적 약점 + 4가지 churn 원인 + 4개 비즈니스 모델 옵션(A 임상 타겟·B 코치 매칭·C B2B·D 시즌제) 제기. **결제 시스템 부착은 의제 아님**, 평가 + 액션 후보 도출만.

## 1. 비평 내용 정직 평가

### 받아들임 (강한 점)

1. **"유저 성공 = 떠남" 미션·비즈니스 모순**: BlueBird 3단계(탐지·시스템2·자율성)의 종착이 self-sufficiency. SDT 정합 그 자체이며 의도된 trade-off.
2. **입력 비용 비대칭**: Calm 버튼 vs BlueBird 쓰기·생각하기. 피곤할수록 안 함 — 실측 가능 영역.
3. **지연 보상 구조**: 명상 즉각 진정 vs BlueBird 3개월 후 패턴 인식. 행동경제학 기본.
4. **프라이버시 부담**(한국 직장인): 도메인 특성상 현실적.

### 보류 또는 거부 (약한 점)

1. "3개월 후 이탈 80%" 정량 단언 — 데이터 0인 추측. IM.1 자연 retention(D44~D74) 실측 대기.
2. "2달쯤이면 다 외워짐" 졸업 가설 — 12 trigger × 5 archetype × 매 사고 사례별 분석은 static 아님. 과장 가능성.
3. "솔로 앱 D90 낮음 → 사회적 요소" — `2026-05-16 deep-dive` 결정잠금에서 ranking·친구 공유·외부 비교 모두 본질 위협 #1 거부. 우회 채택 X.

### 옵션 A~D 평가

| 옵션 | 평가 | 결정잠금 충돌 |
|---|---|---|
| A. 임상 타겟 좁힘 | ❌ 거부 | `legal-review-2026-05-04 §4` 의료 어휘·진단명 타겟팅 금지 |
| B. 인간 코치 매칭 | ❌ 거부 | `2026-05-16 deep-dive` "인간 코치 매칭 금지" 직격 |
| C. B2B | 🟡 IM.2 시야 검토 | 본질 충돌 0. 트로스트 사례. 단 IM.1 후 |
| D. 시즌제 부트캠프 | 🟢 흥미 — IM.1 후 평가 | 본질 3단계 정합 ("졸업" = 미션 성공) |

## 2. 페르소나별 입장 요약

### CPO
- 동의: 입력 비용·지연 보상 = 실제 약점.
- 거부: A·B = 결정잠금 위반.
- 핵심 가설: "졸업" 우려는 본인 아키타입 외 매 사고 사례는 새로움이지만 **새로움이 인사이트 깊이로 이어지는지**가 실측 필요. P0-1 종단 패턴 카드·P0-2 복합 왜곡 헤드라인이 이 가설 해결 시도였음.

### CSO
- 미션·비즈니스 모순은 의도된 윤리적 정합 — 결함 아님.
- "졸업 정합형 비즈니스 모델"로 재정의 가능성 → 옵션 D 시즌제와 정합.
- B2B(C)는 개인 PMF 검증 전 weakness 노출 위험. IM.2 후.

### CMO
- 외로움 우회: 사회적 요소 추가 X, **자발 추천 측정**으로 사회적 효과 신호 캡처 (서면 리포트 B3에 이미 포함).
- 가격 19,900원 진입장벽 평가 동의. 한국 멘탈헬스 카테고리 baseline 미보유 — 서면 리포트 B2 cross-check 가 1차.
- Calm/Headspace 직접 비교는 카테고리 차이로 무효. 한국 솔로 인지·기록 앱 벤치마크 별도 search 가치.

## 3. 3명 합의

### 받아들임
1. 입력 비용·지연 보상 = 실제 약점. UX·measurement 개선 여지.
2. 미션·비즈니스 모순 = 의도된 정합, 무릅써야 함.
3. IM.1 자연 retention(D44~D74)이 "3개월 이탈" 가설 1차 실측.
4. 시즌제(D) feasibility는 IM.1 후 정식 검토.

### 거부 (결정잠금 재확인)
1. 임상 타겟 좁힘(A) — 톤·legal 충돌
2. 인간 코치 매칭(B) — 본질 결정잠금 위반
3. 사회적 요소(친구·랭킹·공유) — 본질 위협 #1
4. Calm/Headspace retention 직접 비교 — 카테고리 차이

### 보류 (IM.1 데이터 후)
1. 시즌제 모델(D) feasibility
2. B2B(C) 시장 진입
3. 입력 마찰 줄이기 UX 개선 — 측정 baseline 기반 결정

## 5. CEO 종합

비평가의 (1) 미션·비즈니스 모순 통찰은 핵심을 짚었으나 **의도된 trade-off** 입니다. 그러나 그 trade-off가 수익 모델 재설계를 요구할 수 있다는 시즌제(D) 방향은 IM.1 데이터 후 진지하게 검토.

**(2) 4가지 churn 원인 중 입력 비용·지연 보상은 실측 가능한 약점** — 측정 인프라(액션 ①)부터 베타 시작 전 구축.

**(3) 옵션 A·B 단호 거부, C·D 데이터 후 검토**.

비평가의 결론 **"PMF는 가능, BMF는 별개"** — 우리도 동의. IM.1 + 베타 종료 후 30일 retention 모인 시점에 BMF 재논의가 정상 시퀀스.
