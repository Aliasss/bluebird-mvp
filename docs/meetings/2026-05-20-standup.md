- 변경 후: `log_view` 추가 → 입력 비용 측정 인프라 ✓

Commit에 운영 분석 SQL 포함 (Supabase editor 전용):
```sql
SELECT a.user_id,
       EXTRACT(EPOCH FROM (l.created_at - a.created_at)) AS input_seconds
FROM analytics_events a
JOIN logs l ON l.user_id = a.user_id
WHERE a.event_name='log_view'
  AND l.created_at > a.created_at
  AND l.created_at < a.created_at + INTERVAL '1 hour'
GROUP BY a.user_id, a.created_at, l.created_at;
```

이 SQL이 IM.1 사용자 모집 후 입력 소요 시간 분포 분석의 기반이 됨. 현재 데이터 0건 (베타 사용자 없음).
