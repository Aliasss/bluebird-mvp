## 데이터 저장

### Supabase 저장 로직
```typescript
const { data, error } = await supabase
  .from('logs')
  .insert({
    user_id: user.id,      // 자동 연결
    trigger: trigger.trim(),
    thought: thought.trim(),
  })
  .select()
  .single();
```

### 에러 처리
- 인증 만료: 로그인 페이지로 리다이렉트

- 데이터베이스 오류: 재시도 안내

## 테스트 방법

### 환경 변수 미설정 시 (현재 상태)
1. http://localhost:3000 접속
2. "시작하기" → 회원가입 시도
3. 환경 변수 오류 발생 (예상됨)
4. UI는 정상 작동하지만 데이터 저장 불가

### Supabase 설정 완료 시
1. 회원가입/로그인
2. 대시보드에서 "기록 시작하기" 클릭
3. 트리거 입력 (예: "상사가 내 의견을 무시했다")
4. "다음" 클릭
5. 자동 사고 입력 (예: "나는 무능하다")
6. "분석 시작" 클릭
7. 분석 페이지로 이동 (로딩 표시)
8. 대시보드 복귀 → 통계 업데이트 확인
9. 최근 활동에 로그 표시 확인
