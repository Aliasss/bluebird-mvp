# 자율성 지수 시계열 고도화 설계 문서

**날짜**: 2026-04-17  
**범위**: /insights 페이지 — 기간 필터 + 성장 지표 비교

---

## 목표

현재 30일 고정인 인사이트 페이지에 기간 토글과 전기간 비교 지표를 추가해, 사용자가 자신의 인지 개선 추세를 데이터로 확인할 수 있게 한다.

---

## A. 기간 필터

**위치**: 페이지 상단 (헤더 바로 아래, 콘텐츠 영역 최상단)

**옵션**: `7일 | 30일 | 전체`  
**기본값**: `30일`

**영향 범위**: 필터 값이 바뀌면 아래 세 차트 모두 재계산
- 왜곡 유형 분포 BarChart
- 자율성 지수 추이 LineChart
- 왜곡 강도 RadarChart

**구현**: `period` state (`'7d' | '30d' | 'all'`) 하나로 관리.  
`'all'`인 경우 `since` 날짜 조건 없이 전체 조회.

---

## B. 성장 지표 카드 (기간 비교)

**위치**: 기간 필터 아래, 기존 요약 카드(3개) 위

**구성**: 3개 비교 카드 (가로 배치)

| 카드 | 내용 |
|------|------|
| 평균 왜곡 강도 | 현재 기간 avg vs 직전 동일 기간 avg → ▼ -12% or ▲ +8% |
| 행동 완료율 | 현재 기간 vs 직전 동일 기간 → delta % |
| 가장 개선된 왜곡 | 빈도가 가장 많이 줄어든 왜곡 유형명 (또는 "데이터 부족") |

**기간별 비교 대상**:
- `7일`: 직전 7일 (8~14일 전)
- `30일`: 직전 30일 (31~60일 전)
- `전체`: 비교 불가 → 카드 비활성화 ("전체 기간 선택 시 비교 불가")

**색상**:
- 개선 (강도 ↓, 완료율 ↑): `text-green-600`
- 악화: `text-red-500`
- 변화 없음 / 데이터 부족: `text-text-secondary`
